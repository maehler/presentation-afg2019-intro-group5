Machine learning of leaf shape GWAS

A functional genomics project

Leaf shape in aspen

  • Large natural variation
  • Highly heritable
  • Complex trait

Traditional GWAS

We have tried doing traditional GWAS, but we have too few samples given the complexity of the trait. Also, we have only done univariate GWAS, that is, only considered one SNP at a time.

Why machine learning?

Can we find combinations of SNPs that explain the trait variation better than single SNPs?

Also, could it be possible to integrate other types of data, for example gene expression, to better describe the phenotypic variation?

Data

  • Gene expression: ~35,000 genes, 80 genotypes (htseq-count)
  • Genetic variation: ~4 million SNPs (VCF format)
  • Phenotype data: leaf shape (BLUPs)
    • Area
    • Indent depth
    • Circularity
  • Gene annotations (gff3 format)

Software

  • bcftools (working with VCF files; command line)
  • bedtools (genome arithmetic; command line)
  • plink (file formats SNPs; command line)
  • ranger (random forests; command line/R-package)
  • Downstream analysis in R or Python (or any other language you like)
